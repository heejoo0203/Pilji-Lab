# Release Notes - v2.2.1

- Release date: `2026-03-07`
- Branch: `main`
- Previous: `v2.2.0`

## Summary
v2.2.1은 구역 분석 저장 흐름과 이용내역 관리 UX를 안정화한 릴리즈다.

## Changes
- 구역 분석 결과 자동 저장 제거
- `구역 저장` 동작으로 영속 저장 분리
- 저장 구역 패널을 지도 우측 영역으로 재구성
- 조회기록 다중 선택 삭제 추가
- 구역 분석 중 `uuid` import 누락으로 발생하던 500 오류 수정

## Data
- 구역 저장 결과는 `zone_analyses`, `zone_analysis_parcels`에 저장된다.
- 이용내역 선택 삭제는 `POST /api/v1/history/query-logs/delete`로 처리한다.
