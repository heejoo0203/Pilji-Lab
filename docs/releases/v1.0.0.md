# Release Notes - v1.0.0

- Release date: `2026-03-02`
- Branch: `main`

## Summary
v1.0.0은 필지랩의 첫 운영 배포 기준이다.  
개별 조회, 인증, 파일 처리, VWorld 우회 프록시까지 실제 운영에 필요한 최소 경로를 갖춘 릴리즈다.

## Included
- 지번/도로명 개별 조회
- 연도별 공시지가 이력 조회
- 회원가입, 로그인, 로그아웃, 내 정보 조회
- 파일 업로드, 비동기 처리, 결과 다운로드
- VWorld 직접 호출 실패 시 프록시 재시도

## Runtime
- Web: Vercel
- API: Railway
- Database: Railway PostgreSQL
- Proxy: AWS EC2
