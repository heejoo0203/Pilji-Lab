# Release Notes - v2.2.0

- Release date: `2026-03-05`
- Branch: `main`
- Previous: `v2.1.1`

## Summary
v2.2.0은 인증 흐름, 지도 조회, 정책 페이지, Android wrapper 배포 경로를 정리한 릴리즈다.

## Changes
- 회원가입 이메일 인증과 아이디/비밀번호 찾기 추가
- 지도 클릭 조회, 주소 검색, 연도별 상세, CSV 내보내기 정리
- 조회기록 필터와 정렬 강화
- 개인정보처리방침과 계정삭제 안내 페이지 추가
- Android APK/AAB 배포 경로 정리

## Notes
- Kakao Maps 도메인 등록이 완료돼야 지도 렌더링이 정상 동작한다.
- VWorld 네트워크 제약이 있는 환경에서는 프록시 구성이 필요할 수 있다.
