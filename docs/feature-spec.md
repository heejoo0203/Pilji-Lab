# 기능 상세 명세 (현재 상태)

## 1. 인증
상태: 구현 완료

### 제공 기능
- 회원가입
- 로그인
- 로그아웃
- 현재 사용자 조회(`me`)

### 핵심 정책
- HttpOnly 쿠키 인증
- 비밀번호 정책 검증(길이, 조합, 바이트 길이)
- 이메일 중복 방지

## 2. 개별조회(지번/도로명)
상태: 구현 완료

### 지번 조회
- 입력: 시/도, 시/군/구, 읍/면/동, 산구분, 본번, 부번
- 처리: PNU 생성 -> VWorld 개별공시지가 조회

### 도로명 조회
- 입력: 시/도, 시/군/구, 자음, 도로명, 건물번호
- 처리: 도로명 -> 좌표 -> 지번 -> PNU -> 공시지가 조회

### 결과 처리
- 연도별 최신 데이터 선택
- 최신연도 내림차순 정렬
- 사람이 읽는 주소 요약 생성

## 3. 도로명 선택기
상태: 구현 완료

### 동작
- 시/군/구 선택 후 해당 지역의 사용 가능한 자음만 표시
- 자음 선택 후 해당 자음으로 시작하는 도로명만 표시
- 데이터 소스: `docs/TN_SPRD_RDNM.txt`

## 4. 조회기록
상태: 1차 구현 완료(클라이언트 저장)

### 동작
- 로그인 사용자 기준으로 기록 분리 저장(ownerKey)
- 최근순 목록 표시
- 목록 클릭 시 해당 조회 결과 재열람

### 제약
- 브라우저 localStorage 기반으로 기기/브라우저 변경 시 동기화되지 않음
- 서버 영구 저장 기능은 다음 단계

## 5. 파일조회
상태: UI만 구현

### 현재
- 파일조회 페이지, 빈 상태/작업상태 UI 제공

### 미구현
- 업로드 API
- 비동기 처리(Celery/Redis)
- 다운로드 API
- 서버 작업 이력 저장

## 6. UI/UX
상태: 구현 완료

### 반영 사항

- 모달 드래그 선택 시 오동작 닫힘 방지
